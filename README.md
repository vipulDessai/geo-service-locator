# Geo Service Locator
Convert human-readable location to computer-readable location n cache them for further analysis 

# Mongo DB query import data
```shell
$ mongoimport --uri "mongodb+srv://root:<password>@cluster0.rycwn.mongodb.net/geo-service-locator" --collection restaurants --drop --file /e/Documents/Repositories/geo-service-locator/data/restaurants.json
```
